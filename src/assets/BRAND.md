# Brand assets

Per the CG house style: brand assets are **imported as modules** from this
folder; `publicDir` is disabled in `vite.config.ts`, so nothing is served
from a public directory. These are the shared CGOPS platform marks, reused
here so New Restaurant Center matches the other applications.

- `CG Logo Small.png` — the boxed CG monogram (181×89). Used in the app
  shell nav rail and the CGOPS redirect screen. Rectangular (~2:1): size it
  by height with automatic width (`h-6 w-auto`), never as a square.
- `CG Logo.png` — the Charcoal Group Restaurants wordmark (1340×146).
  Available for future surfaces; not currently placed in the app.
